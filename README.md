# Portfolio Website

A personal portfolio website built with Next.js 15 and Tailwind CSS.

## Features

- About section
- Services section
- Work portfolio display
- Contact section
- Responsive design

## Technologies

- Next.js 15.x
- React 19
- Tailwind CSS

## Getting Started

Run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
